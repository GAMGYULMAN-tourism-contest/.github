## 📝 소개
### 프로젝트 소개
- ##### 외국인 관광객들을 위한 제주도 여행 계획 협업 플랫폼
    제주도 여행 계획을 체계적으로 관리할 수 있도록 협업 가능한 워크스페이스를 제공하는 플랫폼입니다. 간편한 여행지 검색 기능과 여행 일정 변경의 실시간 반영으로 맞춤형 여행 계획 수립에 도움을 줍니다. 또한 한국 여행 시 자주 사용하는 한국어 표현과 발음을 제공하여 원활한 의사소통을 지원합니다.
